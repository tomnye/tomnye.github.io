Prediction vs interpretation

Contrasting questions:-

  1. Is a protein thermostable, given its sequence and structure?
  2. Which structural elements contribute to thermostability? \[\\[1ex]\]

Statistics vs machine learning:-

  1. Black-box / neural nets will win for prediction accuracy as quantity of data increases
  2. Neural net approaches remain opaque; statistical / model-based approaches win for interpretability

Logistic regression model

  • \(x=\) feature vector for a protein
  • \(p(x)=\) Pr(protein is thermostable) \[\\[1ex]\]

Logistic regression model:-

\[\frac{p(x)}{1-p(x)} = e^{\{\beta_0+\langle \beta,x \rangle\}}\] where \(\beta\) is a vector of coefficients, \(\langle \beta,x \rangle =\) vector inner product \[\\[1ex]\]

Model fitting:-

  • Which coefficients are non-zero (significance)?
  • Which coefficients are largest?

Geometrical interpretation

  • Coefficients \(\beta\) give a direction vector in feature space which best distinguishes between classes
  • Could we change the feature space to a space of graphs?

Red: stable; blue: unstable

Proposed methodology

  • Represent protein as a graph – contact graph
  • Assign attributes to edges and vertices
    • Numerical parameters characterising sequence and structure
    • Select parameters to maintain interpretability
  • Develop logistic regression in graph space
    • Use existing graph space of Callisano and Feragen
    • Contains basic ingredients for regression with graph-valued covariate \[\\[1ex]\]
